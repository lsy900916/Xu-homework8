# XU-News-AI-RAG Scripts

一键脚本工具集

## 脚本列表
- `init.sh/init.bat` - 项目初始化
- `dev.sh/dev.bat` - 启动开发环境
- `build.sh/build.bat` - 构建生产版本
- `test.sh/test.bat` - 运行测试
- `lint.sh/lint.bat` - 代码检查
- `deploy.sh/deploy.bat` - 部署脚本

## 使用方式
```bash
# Linux/Mac
chmod +x scripts/*.sh
./scripts/init.sh

# Windows
.\scripts\init.bat
```
